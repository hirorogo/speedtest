## 実装済みテンプレート

### B1: CSSアクション
4個のボックスが横並びで、クリック時にハイライト機能を持つレイアウト。

**特徴:**
- 80x80pxのボックス4個
- 中央配置されたテキスト
- クリック時の黄色ハイライト効果
- キーボード操作対応

### C1: キーワードハイライター
テキスト内のキーワードをランダムカラーでハイライト表示する機能。

**特徴:**
- 大文字小文字を区別した検索
- ランダムカラーハイライト
- Enterキーでの検索実行
- Escapeキーでのクリア機能

### C5: ナイトモード
ライトモードとナイトモードの切り替え機能。

**特徴:**
- ワンクリックでのモード切り替え
- localStorage での設定保存
- システム設定の自動検知
- キーボードショートカット対応

## 開発ガイドライン

### コーディング規約
- [コーディング規約](./.github/code-style.md)を参照
- [コメント記述規則](./.github/comment-rule.md)を参照

### 学習重視のアプローチ
1. **理解重視**: コードの動作原理を理解してから実装
2. **エラーハンドリング**: 適切なエラー処理を必ず含める
3. **詳細な説明**: 初学者にも分かりやすいコメントを記述
4. **段階的実装**: 小さな機能から始めて段階的に拡張

### 使用技術
- **HTML5**: セマンティックな構造
- **CSS3**: モダンなレイアウト技術（Flexbox, Grid）
- **JavaScript（ES6+）**: 最新の記法を使用
- **レスポンシブデザイン**: モバイル対応必須

## 練習の進め方

1. **問題文を読む**: `.github/speedtest.md`で要件を確認
2. **計画立案**: 必要な機能とファイル構成を検討
3. **段階的実装**: HTML構造 → CSS装飾 → JavaScript機能
4. **テスト**: 各ブラウザでの動作確認
5. **改善**: パフォーマンスとアクセシビリティの向上

## 注意事項

- 作業時間は1時間を想定
- フォルダ管理を明確に行う
- 実際の競技では素材の配布なし
- サーバにアップロードされたもののみが採点対象
